ShapeType class

An enum class with the different kinds of shapes. The two kinds of valid shapes are circles and polygons.

class ShapeType {
  // Circle and Polygon.
  static const int TYPE_COUNT = 2;

  static const int UNKNOWN = -1;
  static const int CIRCLE = 0;
  static const int POLYGON = 1;
}
