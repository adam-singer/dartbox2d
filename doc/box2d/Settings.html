Settings Class

Global tuning constants based on MKS units and various integer maximums(vertices per shape, pairs, etc.).

Static Methods

Code num mixFriction(num friction1, num friction2) #

Friction mixing law.

static num mixFriction(num friction1, num friction2) {
  return Math.sqrt(friction1 * friction2);
}

Code num mixRestitution(num restitution1, num restitution2) #

Restitution mixing law.

static num mixRestitution(num restitution1, num restitution2) {
  return restitution1 > restitution2 ? restitution1 : restitution2;
}
