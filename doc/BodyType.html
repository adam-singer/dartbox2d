class BodyType

Supertype:

  • Object
A body type enum. There are three types of bodies. Static: Have zero mass, zero velocity and can be moved manually. Kinematic: Have zero mass, a non-zero velocity set by user, and are moved by the physics solver. Dynamic: Have positive mass, non-zero velocity determined by forces, and is moved by the physics solver.
